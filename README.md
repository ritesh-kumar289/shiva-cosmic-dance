# शिव — The Cosmic Dance (Tandava)

A cinematic scroll-driven WebGL experience exploring Shiva's Tandava — the cosmic dance of creation and dissolution.

Built with React Three Fiber, Three.js, drei, postprocessing, Lenis smooth scroll, and the Web Audio API.

## Run locally

```bash
npm install
npm run dev
```

Visit http://localhost:5174/

## Build

```bash
npm run build
npm run preview
```

## Deploy

Configured for Vercel — `vercel.json` sets the framework to Vite and adds long-cache headers for `/audio/*` and `/models/*` static assets.
